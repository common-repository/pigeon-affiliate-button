Pigeon Affiliate Button helps to create amazon or any kinds of affiliate link with image and button both with single link/url.

== Description ==

Pigeon Affiliate Button helps to create amazon or any kinds of affiliate link with image and button both with single link/url. It's can be used by shortcode. [button]

Available attributes
        'height'=>'250',
        'width'=>'200',
        'url'=>'your link',
        'style'=>'primary', // danger, info, success
        'position'=>'bottom or top',
        'text'=>'Button Text',
        'src'=>'image url',
        'align'=>'right', //left
        'target'=>'',
        'rel'=>'nofollow'

Major features include:
* Automatically create button with your link.
* Each button contains image as like your product image. 
* You can change size, url, button color, relation like do follow or no follow.


== Installation ==

Upload the Pigeon Affiliate Button plugin to your blog, activate it, and then use the shortcode [button url='https://example.com' style='primary' text='Buy now' src='images/img.jpg'] 

1, 2, 3: You're done!
